## **SCENARIO:**

An E-commerce brand is losing customers during
the checkout process. The layout is cluttered,
and users are confused. They need a clean,
logical structure for their Landing Page.

## **MISSION:**

- Design the "Skeleton" of the application focusing only on User Flow and Hierarchy.

- Create Low-Fidelity Wireframes for a Homepage.

- Ensure there are clear Call-to-Action (CTA) buttons.

## **NOTE:** 

Use only Black, White, and Grey. No images or colors allowed yet!

## **TOOLS:** 

Figma / Pen & Paper.
