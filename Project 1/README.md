## **SCENARIO:**

A new fitness startup wants to launch an app for busy office professionals who struggle to find time for exercise. Before designing a single pixel, the team needs to understand the "User’s Pain".

## **MISSION:**

- Act as a UX Researcher and define the target audience.

- Create User Personas (Age, goals, frustrations).

- Build an Empathy Map (What do they see, think, feel, and do?).

## **OUTCOME:** 

A clear understanding of the user before moving to design.

## **TOOLS:** 

Figma / FigJam / Canva, etc.
